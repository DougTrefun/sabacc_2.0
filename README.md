# SABACC

A Star Wars Sabacc card game clone using Python and Arcade.

## Setup

1. Install dependencies:
   pip install -r requirements.txt

2. Run the game:
   python -m sabacc.main

## Card Images

Place all card images in `assets/cards/`, including `Back.png`.
